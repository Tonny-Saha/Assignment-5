Q1: What is DOM? What is the purpose of DOM?


Ans:DOM stands for Document Object Model in its entire form. In website development, DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from a website document.


Q2: How to select element in DOM?


Ans:getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll()


Q3: What is event bubble?


Ans: Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.


Q1: What is call break function? How does it work?


Ans:Callback functions in JavaScript are distinct functions that are used as arguments in other functions. The parent function or higher level function calls this callback function, which runs following the completion of the parent function. The majority of asynchronous programming is handled through callback functions in event-driven programming languages like JavaScript. Typically, when making an asynchronous server request using an object, we can supply a callback function that will be called when the server returns object data.
